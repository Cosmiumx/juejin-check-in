# 掘金签到插件

这是一个Chrome浏览器插件，能够自动在掘金平台上进行签到，帮助用户轻松获得矿石奖励。

## 功能特点

- 自动检测今日是否已签到
- 在浏览器启动时自动签到
- 每日定时自动签到
- 手动签到功能
- 签到状态显示
- 隐藏标签页自动签到，无需手动操作

## 安装方法

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目的文件夹路径
5. 插件安装完成

## 使用方法

1. 安装插件后，确保您已在掘金网站登录账号
2. 插件会自动在每天上午9点左右进行签到
3. 也可以点击浏览器工具栏中的插件图标，使用手动签到功能
4. 插件会记录签到状态，避免重复签到

## 技术架构

根据预设的工作流程，插件包含以下组件：

1. **Service Worker**: 后台服务，负责定时任务、签到状态检查和标签页管理
2. **Content Script**: 在掘金页面执行签到操作的脚本
3. **Popup界面**: 用户交互界面，显示签到状态和提供手动签到功能

## 文件结构

```
juejin-check-in/
├── manifest.json         # 插件配置文件
├── service-worker.js     # 后台服务工作脚本
├── content-script.js     # 页面内容脚本
├── popup.html           # 弹窗界面
└── popup.js             # 弹窗交互逻辑
```

## 工作流程

1. Chrome启动 → Service Worker被唤醒
2. Service Worker判断今天是否已签到
3. 如未签到，则打开隐藏标签页
4. Content Script自动点击"签到"按钮
5. 记录签到时间
6. 关闭标签页

## 注意事项

- 需要在掘金网站登录账号才能正常签到
- 由于网站结构可能发生变化，签到功能可能会失效，需要更新选择器
- 插件会在后台自动运行，不会影响正常使用
