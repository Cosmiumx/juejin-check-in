# 掘金签到助手

![掘金签到助手](https://unpkg.com/cosmium@0.0.6/images/other/juejin.png)

一款便捷的浏览器扩展，帮助您在掘金平台完成每日签到任务。支持手动和自动两种签到模式，让您的掘金签到变得轻松简单。

## 功能特性

- ✨ **一键签到** - 点击按钮即可完成每日签到
- 🤖 **自动签到** - 支持自动模式，在每天首次打开浏览器时自动签到
- 👁️ **状态监控** - 实时显示今日签到状态
- ⏰ **定时提醒** - 内置定时机制，确保不错过签到
- 🔔 **记录追踪** - 记录最近一次签到时间

## 安装方法

### 方法一：开发者模式安装

1. 克隆或下载此仓库到本地
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹即可

### 方法二：打包安装

1. 在项目根目录下，压缩所有文件为 `.zip` 文件
2. 在 Chrome 扩展页面，拖拽压缩包到页面完成安装

## 使用说明

### 基本使用

1. 安装完成后，点击浏览器工具栏中的扩展图标
2. ***注意: 确保您已经在掘金网站登录了账户***
3. 在弹窗中可以看到今日签到状态
4. 选择合适的签到模式：
   - **手动模式**：需要手动点击"手动签到"按钮进行签到
   - **自动模式**：每天首次打开浏览器时会自动签到
5. 点击"手动签到"按钮完成签到

### 签到模式

- **手动模式**：需要您主动点击按钮完成签到
- **自动模式**：插件会在每天首次打开浏览器时自动为您签到

### 状态提示

- 绿色背景：今日已签到
- 红色背景：今日未签到
- 灰色背景：正在检查状态

## 使用条件

- 已安装 Chrome 或其他 Chromium 内核浏览器
- 已在掘金网站登录账户
- 授予扩展对 `https://juejin.cn/*` 的访问权限
- 确保网络可以正常访问掘金网站

## 技术架构

此扩展基于 Manifest V3 架构开发，包含以下核心组件：

- **Service Worker**: 后台服务，处理定时签到和模式管理
- **Content Script**: 在掘金页面上执行实际的签到操作
- **Popup**: 扩展弹窗界面，提供用户交互功能
- **Storage**: 存储签到状态和用户设置

## 文件结构

```

├── manifest.json       # 扩展配置文件

├── popup.js           # 弹窗界面逻辑
├── content-script.js  # 页面内容脚本
├── service-worker.js  # 后台服务脚本
├── constants.js       # 常量定义
├── common-actions.js  # 公共动作函数
├── utils.js           # 工具函数
└── icons/             # 图标文件夹
```

## 注意事项

- 首次使用时，需要先访问掘金网站并登录账号
- 自动签到功能仅在浏览器启动时触发
- 插件会在签到完成后自动关闭临时标签页
- 如遇签到失败，可尝试手动签到
